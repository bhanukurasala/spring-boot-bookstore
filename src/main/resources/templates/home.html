Welcome to Book Store

The Book Store Application is a web-based platform developed using Spring Boot that allows users to browse, manage, and organize books efficiently. The application provides a clean and user-friendly interface where users can view available books, add new books, update book details, and remove books when required. It simplifies book management by storing all data in a structured database and displaying it dynamically on web pages.
